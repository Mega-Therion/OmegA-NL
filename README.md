# Jarvis Neuro-Link

A Next.js command center prototype showcasing retrieval-augmented context, memory weaving, and adaptive quality controls.

## Features
- RAG signal layer with lightweight vector similarity matching.
- Memory bank persistence with pinning.
- Performance telemetry and quality mode controls.
- Command-driven UI shortcuts (`/help`, `/memory`, `/rag`, `/quality`).

## Getting Started
```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the interface.
